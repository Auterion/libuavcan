Detailed Description

template<typename Type, typename USecT = std::int64_t>
class libuavcan::duration::Base< Type, USecT >

Protected base class for duration values.

Template Parameters
TypeThe type of the derived class. This must be an empty type. All storage will be provided by this base class.
USecTThe datatype returned when retrieving durations from realizations of this base class. This type must be signed.
