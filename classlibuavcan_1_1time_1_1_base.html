Detailed Description

template<typename Type, typename DType, typename USecT = std::uint64_t>
class libuavcan::time::Base< Type, DType, USecT >

Protected base class for time values.

Template Parameters
TypeThe type of the derived class. This must be an empty type. All storage will be provided by this base class.
DTypeThe type of duration used for this time type. Time is concrete and duration is relative.
USecTThe datatype returned when retrieving time from realizations of this base class. This type must be unsigned.
